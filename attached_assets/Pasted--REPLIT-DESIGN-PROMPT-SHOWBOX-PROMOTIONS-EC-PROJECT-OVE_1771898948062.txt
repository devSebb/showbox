# REPLIT DESIGN PROMPT — SHOWBOX PROMOTIONS EC

## 🥊 PROJECT OVERVIEW

Build a **full-fledged, single-page marketing landing website** for **Showbox Promotions EC**, a professional boxing promotion company based in **Quito, Ecuador**. The site promotes their upcoming event: **"Quorum Quito Fight Night XIII"** sponsored by **Forbet**.

The entire site must be in **Spanish**. Dark mode only. No light mode toggle needed.

I am uploading the company logo (SHOW in white, BOX in red, PROMOTIONS below — on black background) and the official fight card poster image. Use these as the visual DNA for the entire site.

---

## 🎨 DESIGN DIRECTION & VISUAL IDENTITY

### Inspiration / Reference Sites:
- **UFC.com** — the aggressive hero sections, fighter profile cards, dark cinematic UI
- **DAZN.com** — sleek dark streaming interface, bold typography, countdown timers
- **TopRank.com** — clean boxing promotion layout, event-first hierarchy
- **Matchroom Boxing** — editorial photo treatment, premium sponsor integration
- **ONE Championship** — immersive full-bleed imagery, dramatic parallax scrolling

### Brand Colors (extract from logo):
- **Primary Black:** #0A0A0A (backgrounds)
- **Deep Black:** #111111 (card surfaces)
- **Showbox Red:** #E31B23 (accent, CTAs, highlights — the red from "BOX" in the logo)
- **White:** #FFFFFF (primary text)
- **Light Gray:** #A0A0A0 (secondary/body text)
- **Gold/Amber:** #D4A843 (optional premium accent for championship references, VIP sections)

### Typography:
- **Headlines:** Use an ultra-bold, compressed/condensed sans-serif — something like **Anton, Oswald Black, or Bebas Neue**. ALL CAPS. Massive scale. Think UFC broadcast graphics.
- **Subheadings:** Bold weight of a geometric sans-serif like **Montserrat Bold** or **Inter Bold**.
- **Body text:** Clean sans-serif, **Montserrat Regular** or **Inter Regular**, high readability.
- Headlines should feel like they PUNCH you in the face. Enormous, compressed, aggressive.

### Visual Treatment:
- **Cinematic dark atmosphere** — think smoky arena, spotlight beams, dramatic contrast
- Use subtle **CSS grain/noise texture overlay** on dark backgrounds for a raw, gritty fight-night feel
- **Red glowing accents** — subtle red glow/shadow effects on hover states and key elements
- **Diagonal slashes / angular clip-paths** — section dividers should NOT be straight horizontal lines. Use aggressive angled cuts, skewed dividers, or diagonal separations between sections (like a punch cutting through the page)
- **Parallax scrolling** on hero and key sections
- Smooth **scroll-triggered animations**: elements slide in, scale up, or fade in as user scrolls
- **Fighter photos** should have a duotone or high-contrast black & white treatment with red color splash accents
- Subtle **particle effects or floating ember/spark animations** in the hero section background

## ⚡ ANIMATIONS & INTERACTIONS

- **Hero countdown timer:** Real-time countdown to March 21, 2026
- **Scroll-triggered reveals:** Every section fades/slides in on scroll (use Intersection Observer or a library like AOS/Framer Motion)
- **Fighter cards:** Staggered entrance animation, hover lift + red glow
- **Stats counters:** Animate counting up from 0 when scrolled into view
- **Navbar:** Transparent on top, gains dark blur background on scroll
- **CTA buttons:** Scale up slightly on hover + red box-shadow glow pulse
- **Smooth scrolling:** All anchor links smooth scroll to their sections
- **Subtle background grain:** CSS noise texture overlay on all dark sections for cinematic grit
- **Diagonal section dividers:** Use CSS clip-path or SVG for aggressive angular transitions between sections

## 📱 RESPONSIVE DESIGN

- **Desktop-first** but must be fully responsive
- Mobile: stacked layout, hamburger menu, fight cards stack vertically, countdown timer wraps to 2x2 grid
- Tablet: adjust grid to 2 columns where needed
- All text must remain readable at all breakpoints
- Touch-friendly button sizes on mobile (minimum 48px tap targets)

## 🎯 KEY DESIGN PRINCIPLES

1. **IMPACTO VISUAL** — Every section should feel like a punch. Bold, dramatic, unapologetic.
2. **DARK & CINEMATIC** — Like walking into a boxing arena at night. Smoke, spotlights, shadows.
3. **RED AS POWER** — Red is used sparingly but powerfully — CTAs, accents, "VS", key highlights.
4. **TYPOGRAPHY IS THE WEAPON** — Massive compressed headlines carry the energy. Type IS the design.
5. **PROFESSIONAL & PREMIUM** — This is NOT a garage gym flyer. This is a world-class promotion. Clean spacing, intentional hierarchy, polished interactions.
6. **MOBILE-FIRST ENERGY** — Even on a phone, this site should make you want to buy a ticket.
